Manfredor
=========

Automatic classification of object based on tags.

Dependencies:
============

* python
* scipy
* numpy

How to:
=======

  git clone --recursive https://github.com/sovannara-hak/manfredor.git
  cd manfredor
  python setup.py install --prefix=/valid/python/path
  
Test:
=====

  python test/test.py
